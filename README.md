# Nexus (Public Shell)

Welcome to the **Nexus Shell**. This repository contains the UI/UX implementations for Android (Kotlin) and Desktop (Electron).

## The Nexus Philosophy
- **Elegance**: A premium, minimalist interface designed for focus.
- **Eco-Friendly**: Intelligent thermal monitoring to protect your hardware.
- **Privacy-First**: No cloud, absolute data ownership.

## Tech Stack
- **Android**: Kotlin, Room (Encrypted), Ktor, ML Kit.
- **Desktop**: Electron, Node.js, Better-SQLite3, Express.
- **Core**: Anti-Gravity Sharding Engine (Google Agentic Architecture).

---
*Note: The core logic (Private Engine) is injected during build-time via the Shield configuration.*
